Prompt ( GEMINI )

buat sebuah program dengan bahasa java dan python, menggunakan struktur data array (fixed-size arrray) dengan kapasitas 10, untuk menyimpan sejumlah data mahasiswa yaitu nama, nim dan kelas. program memiliki menu sebagai berikut : 1.insert at beginning, 2.insert at given position, 3.insert at end, 4.delete from beginning, 5.delete given position, 6.delete from end, 7.delete first occurence, 8.show data, 9.exit. Hal ang harus diperhatikan adalah  :  
1. setiap operasi insert maka variabel count ditambah dengan 1.
2. Setiap operasi delete variabel count dikurang dengan 1
3. kapasitas array baik baris maupun kolom boleh ditambah jumlahnya
4. tidak boleh mengunakan library untuk operasi array
